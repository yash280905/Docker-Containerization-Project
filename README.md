# Docker-Containerization-Project

This project demonstrates deploying a full-stack Java application using containerization and cloud infrastructure. The environment is provisioned on AWS EC2 and orchestrated using Docker and Docker Compose, enabling scalable, modular, and maintainable service deployment.
Key Features

Provisioned an AWS EC2 instance to host a production-ready Java full-stack application.

Containerized multiple services including Nginx, Tomcat, MySQL, RabbitMQ, and Memcached using Docker & Docker Compose for seamless orchestration.

Configured custom Docker networks, ensured service communication, and implemented persistent volumes for reliable data storage.
Technologies Used

AWS EC2 | Docker | Docker Compose | Linux | Java | MySQL | Nginx | RabbitMQ
